== Description ==

Support to change the links in article content. Links from e-commerce sites in the article content will be changed to links through markets or affiliate platforms in Vietnam. 

Hỗ trợ thay đổi các liên kết trong nội dung bài viết. Liên kết từ các trang thương mại điện tử trong nội dung bài viết sẽ được chuyển thành liên kết thông qua các chợ hoặc nền tảng affiliate tại Việt Nam.

Các nền tảng được hỗ trợ bao gồm:
- Lazada Vietnam Affiliate Program: https://www.lazada.vn/lazada-affiliate-program/
- Shopee Vietnam Affiliate Program: https://shopee.vn/affiliate/
- AccessTrade nền tảng tiếp thị liên kết: https://accesstrade.vn/
- Masoffer nền tảng Affiliate marketing: https://masoffer.com/
- Ecomobi Social selling platform: https://ecomobi.com/vi/


== Installation ==

This section describes how to install the plugin and get it working.
1. Unzip the plugin contents to the `/wp-content/plugins/viet-affiliate-link/` directory
2. Activate the plugin through the \\\'Plugins\\\' menu in WordPress.
